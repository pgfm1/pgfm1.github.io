These are my current technical projects:

meager.io

html, web

This website is a small personal project. It allows me to write in a non-professional setting and encourages me to play with more small exercises and projects. The site is deliberately starting in an extremely minimal format with semantic HTML and a very small amount of CSS. It is 100% static, and initially manually updated.

Long-term I'm interested in building out my own simple static site generator so that I can do some basic templating, and write my articles using Markdown rather than HTML. I'm also interested in having that generator do aggregation based on tags, and potentially exploring features based on that.

meager-auth

scala, security, experiment

Intended to be used as a module for other personal projects. I've managed to become completely indecisive as to how I structure this project, despite it not being particularly difficult. This project currently serves as a type of technical playground.
